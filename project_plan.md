# Ahmed Magdy Anwar - Counseling Psychologist

## 1. Project Description
Personal brand website for Ahmed Magdy Anwar, a Counseling Psychologist based in Egypt. The site aims to build trust, attract clients, and make booking sessions easy. Target audience: individuals seeking mental health support, personal growth training, and corporate wellness workshops. The design must feel professional, safe, emotionally calming, and trustworthy.

## 2. Page Structure
- `/` - Home Page (Hero, About Preview, Services, Why Choose, Testimonials, CTA)
- `/about` - About Ahmed (Full bio, credentials, experience, mission)
- `/services` - Services List (Individual Counseling, Training, Workshops)
- `/booking` - Booking Form (Full Name, Phone, Email, Session Type, Date, Message)
- `/contact` - Contact Page (Phone, Email, Social Media, WhatsApp, Map)

## 3. Core Features
- [ ] Bilingual support: English (default) + Arabic (RTL)
- [ ] Language switcher in navigation
- [ ] Floating WhatsApp button on all pages
- [ ] Responsive design (mobile-first)
- [ ] Booking form with confirmation
- [ ] Smooth animations and transitions
- [ ] SEO optimized pages
- [ ] Professional typography and calming color palette
- [ ] Trust elements (privacy statements, credentials display)

## 4. Data Model Design
No database required for MVP. Booking form will use form submission service.

## 5. Backend / Third-party Integration Plan
- No Supabase needed for initial phase
- No Shopify needed
- No Stripe needed
- WhatsApp integration via wa.me deep link
- Form submission via Readdy form service

## 6. Development Phase Plan

### Phase 1: Design System + Navigation + Home Page
- Goal: Establish the visual design system, navigation, and complete home page
- Deliverable: Home page with all sections, navbar, footer, RTL support, WhatsApp button

### Phase 2: About Page + Services Page
- Goal: Build the About and Services pages with content
- Deliverable: Fully content-rich About and Services pages

### Phase 3: Booking Page + Contact Page
- Goal: Build the booking form and contact page
- Deliverable: Functional booking form, contact page with map and details

### Phase 4: Polish & Enhancements
- Goal: Add animations, testimonials, final SEO touches
- Deliverable: Smooth animations, testimonials section, meta tags, final responsive testing
